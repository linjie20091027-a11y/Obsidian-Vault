# 什么是人工智能

## 定义

**人工智能**（Artificial Intelligence，简称 **AI**）是计算机科学的一个分支，旨在创建能够模拟、延伸和扩展人类智能的系统。简单来说，就是让机器能够像人一样**感知、推理、学习和决策**。

## 核心能力

人工智能系统通常具备以下一种或多种能力：

| 能力 | 说明 | 示例 |
|------|------|------|
| **感知** | 通过传感器或数据理解环境 | 图像识别、语音识别 |
| **推理** | 基于已知信息进行逻辑推导 | 棋类对弈、数学证明 |
| **学习** | 从数据中自动提取规律 | 推荐算法、模式识别 |
| **决策** | 在不确定条件下做出最优选择 | 自动驾驶、智能调度 |
| **自然语言处理** | 理解与生成人类语言 | 翻译、对话机器人 |

## 发展历程

- **1956 年** — 达特茅斯会议，AI 概念正式提出
- **1960s-1970s** — 符号主义主导，专家系统兴起
- **1980s-1990s** — 机器学习逐渐发展，神经网络复兴
- **2006 年** — 深度学习概念提出（Hinton）
- **2012 年** — AlexNet 在 ImageNet 比赛中取得突破，深度学习时代开启
- **2017 年** — Transformer 架构提出，奠定大语言模型基础
- **2022 年至今** — GPT 系列、扩散模型等生成式 AI 爆发

## 主要流派

1. **符号主义（Symbolism）** — 基于逻辑和符号推理，认为智能源于符号操作
2. **连接主义（Connectionism）** — 基于神经网络，认为智能源于神经元之间的连接
3. **行为主义（Behaviorism）** — 基于感知-动作循环，认为智能在与环境交互中涌现

## 技术分类

### 按能力层次

- **弱人工智能（ANI）**：专注于单一任务（如人脸识别、语音助手）—— 当前主流
- **强人工智能（AGI）**：具备与人类相当的通用智能 —— 尚未实现
- **超人工智能（ASI）**：在所有领域超越人类的智能 —— 理论概念

### 按技术路线

```
人工智能
├── 机器学习（Machine Learning）
│   ├── 监督学习（Supervised Learning）
│   ├── 无监督学习（Unsupervised Learning）
│   ├── 半监督学习（Semi-supervised Learning）
│   └── 强化学习（Reinforcement Learning）
├── 深度学习（Deep Learning）
│   ├── 卷积神经网络（CNN）
│   ├── 循环神经网络（RNN / LSTM）
│   ├── Transformer / 注意力机制
│   └── 生成对抗网络（GAN）
└── 生成式 AI（Generative AI）
    ├── 大语言模型（LLM）
    ├── 扩散模型（Diffusion Model）
    └── 多模态模型
```

## 关键技术

| 技术 | 描述 |
|------|------|
| **大语言模型（LLM）** | GPT、Claude、Llama 等，基于海量文本训练的对话与推理模型 |
| **计算机视觉（CV）** | 图像分类、目标检测、图像分割、人脸识别 |
| **自然语言处理（NLP）** | 文本分类、情感分析、机器翻译、问答系统 |
| **语音技术** | 语音识别（ASR）、语音合成（TTS） |
| **强化学习（RL）** | 智能体通过与环境交互获得奖励信号，学习最优策略 |
| **知识图谱** | 用图结构表示实体之间的关系，支持推理与问答 |

## 应用场景

- **医疗健康** — 医学影像分析、药物研发、辅助诊断
- **金融** — 风险评估、欺诈检测、量化交易
- **教育** — 个性化学习、自动批改、智能辅导
- **交通** — 自动驾驶、交通流量预测、智能调度
- **制造业** — 质量检测、预测性维护、智能机器人
- **创意领域** — AI 绘画、音乐生成、视频生成、写作辅助

## 挑战与思考

1. **可解释性** — 深度学习模型往往是"黑箱"，难以解释其决策过程
2. **数据与隐私** — 模型训练依赖大量数据，可能涉及隐私泄露
3. **公平性与偏见** — 模型可能继承训练数据中的社会偏见
4. **安全与对齐** — 确保 AI 系统的行为符合人类意图和价值观
5. **就业影响** — 自动化可能替代部分工作岗位，但也创造新机会
6. **能源消耗** — 大模型训练和推理需要大量算力与电力

## 常用框架与工具

- **PyTorch** — Meta 开源的深度学习框架（当前主流）
- **TensorFlow** — Google 开源的深度学习框架
- **Hugging Face** — 模型库与工具集，涵盖 NLP、CV 等
- **LangChain / LlamaIndex** — 大语言模型应用开发框架
- **scikit-learn** — 经典机器学习库
