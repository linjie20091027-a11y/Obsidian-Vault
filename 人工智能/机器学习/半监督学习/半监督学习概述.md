# 半监督学习概述（Semi-Supervised Learning）

## 一、核心思想

半监督学习（Semi-Supervised Learning, SSL）的核心思想是：**利用少量有标注数据（labeled data）配合大量无标注数据（unlabeled data）来提升模型的学习效果**。

在现实场景中，获取标注数据往往代价高昂（需要人工标注、专家知识），而无标注数据则大量存在且易于获取。半监督学习试图从无标注数据中挖掘数据分布的结构信息，从而在标注样本不足时仍能训练出性能良好的模型。

形式上，给定标注数据集：

$$D_l = \{(x_1, y_1), (x_2, y_2), ..., (x_l, y_l)\}$$

以及无标注数据集：

$$D_u = \{x_{l+1}, x_{l+2}, ..., x_{l+u}\}$$

其中 $l \ll u$（标注数据远少于无标注数据）。目标是学习一个函数 $f: \mathcal{X} \to \mathcal{Y}$，同时利用 $D_l$ 和 $D_u$。

## 二、三大基本假设

半监督学习的有效性建立在以下三大假设之上：

### 2.1 平滑假设（Smoothness Assumption）

如果两个样本 $x_1$ 和 $x_2$ 在高密度区域中是**相近的**（由某个距离度量或图中的边连接），则它们对应的标签 $y_1$ 和 $y_2$ 也应该是相近的。

> 直观理解：高密度区域中的邻近点应属于同一类别。决策边界不应穿过高密度区域。

### 2.2 聚类假设（Cluster Assumption）

数据倾向于形成**聚类结构**，同一聚类中的样本更可能具有相同的标签。

> 与平滑假设密切相关：如果两个点属于同一个聚类（即可以通过高密度路径连接），它们很可能共享同一标签。这是低密度分离（low-density separation）的基础。

### 2.3 流形假设（Manifold Assumption）

高维数据通常分布在某个**低维流形**上。如果两个样本在流形的内在几何结构中是相近的，则它们的标签也应当相近。

> 这一假设允许使用图拉普拉斯、流形正则化等方法，利用无标注数据来估计数据流形的几何结构。

## 三、主要方法分类

### 3.1 自训练（Self-Training）

自训练是最简单直观的半监督方法：

**算法流程：**

1. 使用标注数据集 $D_l$ 训练初始模型 $f_0$
2. 用 $f_0$ 对无标注数据集 $D_u$ 进行预测
3. 选择**置信度最高**的预测样本，将其伪标签加入训练集
4. 用扩展后的训练集重新训练模型
5. 重复步骤 2-4，直到满足停止条件

**伪代码：**

```
输入: 标注数据 D_l, 无标注数据 D_u
重复:
    f = 在 D_l 上训练模型
    对 D_u 中每个 x 预测: ŷ = f(x)
    选择置信度 > τ 的样本，加入 D_l
    从 D_u 中移除已标注的样本
直到 D_u 为空或无新增样本
```

**优点：** 简单，可与任何基础模型结合。
**缺点：** 容易产生**确认偏差（confirmation bias）**——早期错误会自我强化。

### 3.2 协同训练（Co-Training）

协同训练假设数据存在两个**充分冗余且条件独立的视图（views）**。

**核心思想：** 在两个视图上分别训练两个分类器，每个分类器将其高置信度预测提供给对方作为额外的训练样本。

**算法流程：**

1. 在标注数据上分别训练分类器 $C_1$（使用视图1）和 $C_2$（使用视图2）
2. 每个分类器对 $D_u$ 进行预测，选出最置信的 $k$ 个正例和 $k$ 个负例
3. 将这些样本连同伪标签加入对方的训练集
4. 重新训练，迭代直到收敛

**适用场景：** 网页分类（页面文本 + 锚文本）、多模态数据（图像 + 文本描述）等。

### 3.3 标签传播（Label Propagation）

标签传播是一类基于图的半监督学习方法。构建一个图 $G = (V, E)$，其中节点表示所有样本（标注 + 未标注），边权重 $W_{ij}$ 表示样本 $i$ 和 $j$ 之间的相似度（通常用高斯核 $W_{ij} = \exp(-\|x_i - x_j\|^2 / \sigma^2)$）。

定义**标签矩阵** $Y \in \mathbb{R}^{n \times C}$，其中 $Y_{ic} = 1$ 表示样本 $i$ 的标签为 $c$。

**核心思想：** 每个节点的标签是其邻居标签的加权平均，通过迭代传播。

**迭代公式：**

$$F^{(t+1)} = \alpha S F^{(t)} + (1 - \alpha) Y^{(0)}$$

其中 $S = D^{-1/2} W D^{-1/2}$ 为归一化的图拉普拉斯矩阵，$D$ 为度矩阵，$\alpha \in (0, 1)$ 控制从邻居继承信息的比例。

**闭式解：**

$$F^* = (I - \alpha S)^{-1} Y^{(0)}$$

### 3.4 半监督 SVM（S3VM / TSVM）

半监督支持向量机（Semi-Supervised SVM，也称 Transductive SVM / TSVM）在传统 SVM 的基础上，同时利用无标注数据来寻找**远离高密度区域**的决策边界。

**目标函数：**

$$\min_{w, b, \hat{y}} \frac{1}{2}\|w\|^2 + C_l \sum_{i=1}^{l} \xi_i + C_u \sum_{j=l+1}^{l+u} \xi_j$$

满足约束：

$$y_i(w^T x_i + b) \geq 1 - \xi_i, \quad \xi_i \geq 0 \quad (标注数据)$$
$$\hat{y}_j(w^T x_j + b) \geq 1 - \xi_j, \quad \xi_j \geq 0 \quad (无标注数据)$$

其中 $\hat{y}_j \in \{-1, +1\}$ 是无标注数据的伪标签，也作为优化变量。

**关键直觉：** S3VM 试图寻找这样一个超平面——它不仅最大化分类间隔，而且尽量不穿过数据密集区域（低密度分离）。

**优化难点：** 目标函数非凸，$\hat{y}_j$ 是离散变量。常用方法包括：
- 标签切换（label switching）
- CCCP（凹凸过程）
- SDP 松弛

## 四、半监督深度学习

深度学习的兴起为半监督学习带来了新的可能性。以下介绍几种重要的方法：

### 4.1 Π-Model

Π-Model 的核心是**一致性正则化（Consistency Regularization）**：

对同一个输入 $x$ 应用两次不同的随机增强（如 Dropout、数据增强），得到两个预测输出 $f(x, \theta, \xi_1)$ 和 $f(x, \theta, \xi_2)$，要求两者一致。

**损失函数：**

$$L = \frac{1}{l}\sum_{i=1}^{l} L_{ce}(f(x_i), y_i) + \lambda \frac{1}{l+u}\sum_{i=1}^{l+u} \|f(x_i, \xi_1) - f(x_i, \xi_2)\|^2$$

其中 $L_{ce}$ 为交叉熵损失，$\lambda$ 为一致性损失的权重。

### 4.2 Temporal Ensembling

Temporal Ensembling 是 Π-Model 的改进：不再对同一批数据进行两次前向传播，而是维护每个样本的历史预测的**指数移动平均（EMA）**作为目标。

$$Z_t = \alpha Z_{t-1} + (1 - \alpha) z_t$$
$$\tilde{z}_t = Z_t / (1 - \alpha^t)$$

这种方法将目标的更新分散到不同 epoch 中，提高了计算效率。

### 4.3 Mean Teacher

Mean Teacher 在 Temporal Ensembling 的基础上进一步改进：不使用预测结果的 EMA，而是使用**模型参数的 EMA** 构建教师模型。

$$\theta'_t = \alpha \theta'_{t-1} + (1 - \alpha) \theta_t$$

学生模型 $\theta$ 正常训练，教师模型 $\theta'$ 为学生模型的指数移动平均。一致性损失使学生预测逼近教师预测：

$$L_{cons} = \|f(x, \theta) - f(x, \theta')\|^2$$

### 4.4 FixMatch

FixMatch 是近年来最简洁有效的一种半监督学习方法，巧妙地将**一致性正则化**与**伪标签**结合。

**算法流程：**

1. 对无标注样本，首先施加**弱增强（weak augmentation）**（如翻转、平移）
2. 通过模型预测得到伪标签：$\hat{y} = \arg\max p(y|x_{weak})$
3. 仅保留置信度高于阈值 $\tau$ 的伪标签
4. 对同一样本施加**强增强（strong augmentation）**（如 RandAugment、Cutout）
5. 用交叉熵损失匹配强增强的预测与伪标签

**损失函数：**

$$L_{u} = \frac{1}{\mu B} \sum_{b=1}^{\mu B} \mathbf{1}(\max(q_b) \geq \tau) \cdot H(\hat{q}_b, p_m(y | \mathcal{A}(u_b)))$$

其中：
- $q_b = p_m(y | \alpha(u_b))$ 是弱增强下的预测分布
- $\hat{q}_b = \arg\max(q_b)$ 是伪标签
- $\mathcal{A}(u_b)$ 表示强增强
- $\tau$ 为置信度阈值（通常设为 0.95）
- $H$ 为标准交叉熵

**监督损失：**

$$L_s = \frac{1}{B} \sum_{b=1}^{B} H(y_b, p_m(y | \alpha(x_b)))$$

**总损失：**

$$L = L_s + \lambda_u L_u$$

**关键设计：**
- 弱增强用于生成可靠的伪标签，强增强用于鼓励模型的鲁棒性
- 高置信度阈值 $\tau$ 确保伪标签质量
- 简单高效，不需要额外的教师模型或动量更新

## 五、总结与对比

| 方法 | 核心机制 | 是否基于图 | 计算复杂度 |
|------|----------|------------|------------|
| 自训练 | 迭代伪标签 | 否 | 低 |
| 协同训练 | 多视图互相标注 | 否 | 中等 |
| 标签传播 | 图结构传播 | 是 | 中高 |
| S3VM | 低密度分离 | 否 | 高 |
| Π-Model | 一致性正则化 | 否 | 中等 |
| Mean Teacher | 参数EMA | 否 | 中等 |
| FixMatch | 弱增强→伪标签+强增强 | 否 | 中等 |

## 六、关键挑战

1. **确认偏差（Confirmation Bias）**：错误伪标签的累积会导致模型退化
2. **类别分布偏移**：无标注数据的类别分布可能与标注数据不同
3. **超参数敏感**：伪标签阈值、损失权重等对性能影响显著
4. **数据质量**：无标注数据中的噪声和异常值
5. **评估困难**：缺乏标准的评估协议和基准
