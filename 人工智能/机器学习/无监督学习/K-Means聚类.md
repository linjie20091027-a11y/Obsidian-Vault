# K-Means 聚类

## 1. 核心思想

K-Means 是一种基于**原型**的聚类算法。其核心思想是：将数据集划分为 $K$ 个簇（cluster），使得每个样本到其所属簇的质心（centroid）的**距离平方和**最小化。

直观理解：在特征空间中寻找 $K$ 个"代表点"（质心），每个样本被分配到距离最近的质心所代表的簇中。算法通过迭代优化质心位置，最终达到一个局部最优的划分。

---

## 2. 目标函数：Inertia（惯性/畸变）

K-Means 的目标函数称为 **Inertia** 或 **簇内误差平方和（Within-Cluster Sum of Squares, WCSS）**：

$$
J = \sum_{i=1}^{K} \sum_{x \in C_i} \| x - \mu_i \|^2
$$

其中：
- $K$：簇的数量
- $C_i$：第 $i$ 个簇的样本集合
- $\mu_i = \frac{1}{|C_i|} \sum_{x \in C_i} x$：第 $i$ 个簇的质心（均值向量）
- $\|x - \mu_i\|^2$：样本 $x$ 与质心 $\mu_i$ 之间的欧氏距离平方

目标即为寻找最优的簇划分 $\{C_1, C_2, \ldots, C_K\}$ 和质心 $\{\mu_1, \mu_2, \ldots, \mu_K\}$，使得 $J$ 最小化。

> **等价视角**：最小化 Inertia 等价于最大化簇间距离，即最小化**簇内方差**的同时最大化**簇间方差**。

---

## 3. 算法流程（Lloyd 算法）

标准 K-Means 使用 **Lloyd 算法** 进行迭代优化：

| 步骤 | 操作 | 公式 / 描述 |
|:---:|:---|:---|
| **Step 1** | **初始化** | 随机选择 $K$ 个样本点作为初始质心 $\mu_1^{(0)}, \mu_2^{(0)}, \ldots, \mu_K^{(0)}$ |
| **Step 2** | **分配样本** | 将每个样本 $x$ 分配到与其距离最近的质心所属的簇：$C_i^{(t)} = \{ x : \|x - \mu_i^{(t)}\| \leq \|x - \mu_j^{(t)}\|, \forall j \neq i \}$ |
| **Step 3** | **更新质心** | 重新计算每个簇的质心：$\mu_i^{(t+1)} = \frac{1}{|C_i^{(t)}|} \sum_{x \in C_i^{(t)}} x$ |
| **Step 4** | **收敛判断** | 若 $\mu_i^{(t+1)} = \mu_i^{(t)}$（质心不再变化）或达到最大迭代次数，则停止；否则 $t \leftarrow t+1$，返回 Step 2 |

```
算法伪代码：
────────────────────────────────────────
KMeans(X, K, max_iter):
    随机选择 K 个点作为初始质心 μ₁,...,μₖ
    for t = 1 to max_iter:
        for each x ∈ X:
            将 x 分配给最近质心所在的簇
        for each i = 1 to K:
            用簇 i 中所有样本的均值更新 μᵢ
        if 质心未发生变化:
            break
    return 簇分配和质心
────────────────────────────────────────
```

## 4. 收敛性分析

### 4.1 J 单调递减

每次迭代包含两个步骤，每一步都使目标函数 $J$ 非增：

**分配步骤（Assignment Step）**：固定质心 $\mu_i$，为每个样本 $x$ 选择最近的质心。等价于：
$$
C_i^{(t)} = \arg\min_{C} \sum_{x \in C} \|x - \mu_i\|^2
$$
这一步保证了在固定质心时，$J$ 取得当前条件下的最小值，因此 $J$ 不会增加。

**更新步骤（Update Step）**：固定簇划分 $C_i$，重新计算质心。可以证明，对任意簇 $C$，使其样本平方距离和最小的点是样本均值 $\mu$：
$$
\mu_i^{(t+1)} = \arg\min_{\mu} \sum_{x \in C_i} \|x - \mu\|^2
$$
令导数为零：$\frac{\partial}{\partial \mu} \sum_{x \in C_i} \|x - \mu\|^2 = -2 \sum_{x \in C_i} (x - \mu) = 0 \implies \mu = \frac{1}{|C_i|} \sum_{x \in C_i} x$

因此更新步骤也不会增加 $J$。

### 4.2 有下界保证收敛

由于 $J = \sum \|x - \mu_i\|^2 \geq 0$，目标函数有下界 $0$。结合 $J$ 单调递减，根据**单调有界定理**，算法必然收敛。

### 4.3 局部最优

由于问题是非凸的，Lloyd 算法只能保证收敛到**局部最优解**，而非全局最优解。初始质心的选择对最终结果有显著影响。

## 5. K 值选择

K-Means 要求预先指定簇的数量 $K$，选择合适的 $K$ 是应用中的关键问题。

### 5.1 肘部法则（Elbow Method）

绘制 $J(K)$ 随 $K$ 变化的曲线。随着 $K$ 增大，$J$ 单调递减，但递减速率会逐渐放缓。选择曲线**拐点（elbow）**作为最佳 $K$，即增加 $K$ 带来的收益明显减小的位置。

- 优点：简单直观
- 缺点：有时拐点不明显，主观性较强

### 5.2 轮廓系数（Silhouette Coefficient）

对每个样本 $x$，定义：
- $a(x)$：$x$ 到同簇其他样本的平均距离（**簇内凝聚度**）
- $b(x)$：$x$ 到最近其他簇所有样本的平均距离（**簇间分离度**）

轮廓系数：
$$
s(x) = \frac{b(x) - a(x)}{\max(a(x), b(x))}, \quad s(x) \in [-1, 1]
$$

- $s \to 1$：样本分类合理，远离其他簇
- $s \to 0$：样本处于两个簇的边界
- $s \to -1$：样本可能被错误分类

将所有样本的 $s(x)$ 取平均，得到整体的轮廓系数。选择使平均轮廓系数最大的 $K$。

### 5.3 其他指标

| 指标 | 公式 / 描述 |
|:---|:---|
| **Calinski-Harabasz 指数** | $\frac{\text{Tr}(B_K)}{\text{Tr}(W_K)} \times \frac{N-K}{K-1}$，越大越好 |
| **Davies-Bouldin 指数** | 衡量簇内散度与簇间距离之比，越小越好 |
| **Gap Statistic** | 比较 $J(K)$ 与均匀分布参考数据的期望值 |

## 6. K-Means++ 初始化

K-Means++ 通过改进初始质心的选择来提升聚类质量：

**算法步骤**：
1. 从数据集中**均匀随机**选择第一个质心 $\mu_1$
2. 对于每个样本 $x$，计算其到已选质心的最短距离 $D(x) = \min_{j} \|x - \mu_j\|$
3. 以概率 $P(x) = \frac{D(x)^2}{\sum_{x' \in X} D(x')^2}$ 选择下一个质心（距离已有质心越远，被选中概率越大）
4. 重复步骤 2-3 直至选出 $K$ 个质心
5. 使用标准 K-Means 进行后续迭代

**理论保证**：K-Means++ 的结果与最优解的比值期望为 $O(\log K)$，比随机初始化有显著改进。

## 7. Mini-Batch K-Means

**Mini-Batch K-Means** 是标准 K-Means 的随机优化版本，适用于大规模数据集。

**核心思想**：每次迭代不使用全部数据，而是随机采样一个小批量（mini-batch）来更新质心。更新时采用**学习率衰减**策略：

$$
\mu_i \leftarrow \mu_i + \eta \cdot \frac{\sum_{x \in \text{batch}, x \in C_i} (x - \mu_i)}{|\{x \in \text{batch}, x \in C_i\}|}
$$

其中 $\eta$ 是学习率，通常随时间递减（如 $\eta = 1/t$）。

- **优点**：大幅降低每次迭代的计算量，收敛速度显著快于标准 K-Means，适合海量数据
- **缺点**：结果为近似解，Inertia 略高于标准 K-Means

## 8. 核 K-Means（Kernel K-Means）

当数据在原始空间中**线性不可分**（簇非凸、非线性边界）时，核 K-Means 通过核技巧将数据映射到高维特征空间进行聚类。

**核心思想**：利用核函数隐式计算高维空间中的点积，避免显式计算映射 $\phi$：

$$
K(x_i, x_j) = \phi(x_i)^T \phi(x_j)
$$

在高维特征空间中的距离平方可表示为：
$$
\|\phi(x) - \mu_i^\phi\|^2 = K(x, x) - \frac{2}{|C_i|} \sum_{x_j \in C_i} K(x, x_j) + \frac{1}{|C_i|^2} \sum_{x_j, x_l \in C_i} K(x_j, x_l)
$$

样本分配仅依赖于该距离，无需显式构造 $\phi(x)$。

**常用核函数**：

| 核函数 | 表达式 |
|:---|:---|
| 线性核 | $K(x, y) = x^T y$ |
| 多项式核 | $K(x, y) = (x^T y + c)^d$ |
| RBF / 高斯核 | $K(x, y) = \exp\left(-\gamma \|x - y\|^2\right)$ |

使用 RBF 核的核 K-Means 可以发现**任意形状的非凸簇**，但计算复杂度较高（需维护核矩阵）。

## 9. 局限性

| 局限 | 说明 |
|:---|:---|
| **需要预设 K** | 必须预先指定簇的数量，实际应用中往往未知 |
| **对初始值敏感** | 不同的初始质心可能导致完全不同的聚类结果（即使使用 K-Means++ 也只能缓解） |
| **只能发现球形簇** | 基于欧氏距离的分配使算法倾向于产生凸的、各向同性的球形簇，无法处理非凸簇 |
| **对异常值敏感** | 质心为样本均值，异常值会显著拉偏质心位置 |
| **对尺度敏感** | 不同特征的量纲差异会扭曲距离度量，通常需要提前做标准化/归一化 |
| **维数灾难** | 高维空间中欧氏距离的区分能力下降，聚类效果变差 |
| **只能输出硬分配** | 每个样本只能属于一个簇，无法表达不确定性 |

## 10. 与 GMM（高斯混合模型）的对比

| 维度 | K-Means | GMM |
|:---|:---|:---|
| **簇形状** | 球形（等向性） | 椭圆形（任意协方差） |
| **分配方式** | 硬分配（Hard assignment），每个样本只属于一个簇 | 软分配（Soft assignment），输出每个样本属于各簇的概率 |
| **概率模型** | 非概率模型，最小化平方误差 | 概率生成模型，最大化似然（EM 算法） |
| **参数** | 仅质心位置 | 均值、协方差矩阵、混合系数 |
| **收敛** | Lloyd 迭代，确定性 | EM 算法，可能收敛到局部最优 |
| **不确定性** | 无法表达 | 通过后验概率表达归属不确定性 |
| **本质关系** | GMM 的特例：当所有协方差矩阵 $\Sigma_k = \sigma^2 I$ 且 $\sigma \to 0$ 时，GMM 退化为 K-Means | K-Means 的软分配、概率化推广 |

> 简要概括：**K-Means = GMM + 各簇等方差各向同性 + 硬分配**
