# 条件 GAN 与图像翻译

## 1. 条件 GAN（cGAN）

### 1.1 基本思想

标准 GAN 的生成过程是不可控的——我们只能从随机噪声生成随机的逼真图像。**条件 GAN（Conditional GAN, cGAN）** 由 Mirza 和 Osindero 于 2014 年提出，通过向生成器和判别器注入**条件信息** $c$，使生成过程可控。

生成器和判别器的形式变为：

$$G: (z, c) \rightarrow x$$
$$D: (x, c) \rightarrow [0, 1]$$

其中 $c$ 可以是任何形式的辅助信息：
- 类别标签（如生成特定数字的 MNIST 图像）；
- 文本描述（文本到图像生成）；
- 另一张图像（图像翻译任务）。

### 1.2 目标函数

cGAN 的目标函数是标准 GAN 目标函数的条件化扩展：

$$\min_G \max_D V(D, G) = \mathbb{E}_{x \sim P_{data}}[\log D(x, c)] + \mathbb{E}_{z \sim P_z}[\log(1 - D(G(z, c), c))]$$

条件信息 $c$ 与输入 $x$ 或噪声 $z$ 在判别器和生成器中分别进行拼接，构成联合输入。

## 2. pix2pix：配对图像翻译

### 2.1 问题定义

**图像翻译（Image-to-Image Translation）** 的目标是将输入图像从源域映射到目标域，同时保留底层结构信息。例如：
- 语义标签图 → 真实街景照片；
- 灰度图 → 彩色图；
- 白天场景 → 夜晚场景；
- 边缘图 → 真实物体；
- 航拍图 → 地图。

pix2pix 由 Isola 等人于 2017 年提出，是首个通用的**配对（Paired）图像翻译框架**，即训练数据要求源域和目标域图像一一对应。

### 2.2 损失函数

pix2pix 的损失由两部分组成：**条件 GAN 损失 + L1 重建损失**。

**cGAN 损失**：

$$\mathcal{L}_{cGAN}(G, D) = \mathbb{E}_{x, y}[\log D(x, y)] + \mathbb{E}_{x, z}[\log(1 - D(x, G(x, z)))]$$

其中 $x$ 是输入图像（条件），$y$ 是目标图像，$z$ 是随机噪声。pix2pix 发现噪声 $z$ 对生成多样性的贡献很小（生成器倾向于忽略它），因此实际实现中常以 Dropout 作为随机性的来源。

**L1 损失**：

$$\mathcal{L}_{L1}(G) = \mathbb{E}_{x, y, z}\left[ \|y - G(x, z)\|_1 \right]$$

L1 损失鼓励生成图像在像素级别与目标 $y$ 保持一致，相比 L2 损失更不易产生模糊。

**总损失**：

$$\mathcal{L} = \mathcal{L}_{cGAN}(G, D) + \lambda \cdot \mathcal{L}_{L1}(G)$$

其中 $\lambda$ 是平衡超参数（论文中取 100）。cGAN 损失负责确保生成图像的"真实性"（高频结构），L1 损失负责确保"准确性"（低频结构）。

### 2.3 PatchGAN 判别器

pix2pix 使用 **PatchGAN** 作为判别器架构。与传统判别器输出单个全局真假标量不同，PatchGAN 输出一个 $N \times N$ 的矩阵，**每个元素对应输入图像的一个局部图像块（Patch）的真假判断**。

**设计理念**：
- L1 损失已经能够很好地捕捉低频结构（整体色调、布局）；
- 判别器只需关注**高频细节**（纹理、边缘清晰度）；
- 对每个 patch 独立判别等价于假设像素之间的独立性（马尔可夫随机场），这被证明对纹理和风格建模足够有效。

**架构**：
- PatchGAN 是一个全卷积网络，不含全连接层；
- 典型 patch 大小为 $70 \times 70$（感受野），对应 $30 \times 30$ 的输出矩阵；
- 相比全局判别，PatchGAN 参数更少、训练更快，且可适用于任意尺寸的输入图像。

**PatchGAN 损失**：将 $N \times N$ 个 patch 的判别结果的平均交叉熵作为最终损失。

### 2.4 U-Net 生成器

pixpix 采用 **U-Net** 作为生成器架构。U-Net 是一种编码器-解码器结构，其核心特征是**跳跃连接（Skip Connections）**：

```
输入 x → [Encoder: 卷积下采样 × N] → 瓶颈层 → [Decoder: 转置卷积上采样 × N] → 输出 ŷ
         ↓                                                           ↑
         └────────────── 跳跃连接（concat） ─────────────────────────┘
```

**跳跃连接的作用**：
- 将编码器各层的高分辨率特征直接传递到解码器的对应层；
- 保留低层细节信息（边缘、纹理），避免在经过瓶颈层压缩后信息丢失；
- 对于图像翻译任务，输入和输出共享底层结构（如边缘布局），跳跃连接使信息可以在不经过瓶颈层的情况下"短路"传递。

U-Net 最初是为生物医学图像分割提出的，pix2pix 将其引入图像翻译领域，发现效果显著优于普通的编码器-解码器结构。

## 3. CycleGAN：非配对图像翻译

### 3.1 问题定义

配对图像翻译需要成对的训练数据 $(x_i, y_i)$，但在许多场景下获取这种成对数据成本极高甚至不可行（如将莫奈的画风转换为照片风格——不存在一一对应的"莫奈画 vs 同场景照片"）。

**CycleGAN** 由 Zhu 等人于 2017 年提出，实现了**非配对（Unpaired）图像翻译**，只需两组独立的图片集合：源域 $X = \{x\}$ 和目标域 $Y = \{y\}$，无需任何配对关系。

### 3.2 核心思想：循环一致性

CycleGAN 训练**两个生成器**和**两个判别器**：

$$G: X \rightarrow Y \quad \text{（将源域图像转换到目标域）}$$
$$F: Y \rightarrow X \quad \text{（将目标域图像转换回源域）}$$
$$D_X: \text{判别 } x \text{ 是否为真实源域图像}$$
$$D_Y: \text{判别 } y \text{ 是否为真实目标域图像}$$

核心创新是**循环一致性损失（Cycle Consistency Loss）**：

$$\mathcal{L}_{cycle}(G, F) = \mathbb{E}_{x \sim P_X}\left[ \|F(G(x)) - x\|_1 \right] + \mathbb{E}_{y \sim P_Y}\left[ \|G(F(y)) - y\|_1 \right]$$

**直观理解**：如果将一张照片翻译成梵高风格再翻译回照片，应该得到与原始照片相同的图像。循环一致性约束阻止了 $G$ 和 $F$ 相互矛盾（如所有输入都映射到同一张目标域图像）。

**循环一致性示意图**：
```
x (源域) ──G──→ G(x) ──F──→ F(G(x)) ≈ x     （前向循环）
y (目标域) ──F──→ F(y) ──G──→ G(F(y)) ≈ y   （后向循环）
```

### 3.3 完整损失函数

**对抗损失**：
$$\mathcal{L}_{GAN}(G, D_Y, X, Y) = \mathbb{E}_{y \sim P_Y}[\log D_Y(y)] + \mathbb{E}_{x \sim P_X}[\log(1 - D_Y(G(x)))]$$
$$\mathcal{L}_{GAN}(F, D_X, Y, X) = \mathbb{E}_{x \sim P_X}[\log D_X(x)] + \mathbb{E}_{y \sim P_Y}[\log(1 - D_X(F(y)))]$$

**循环一致性损失**：
$$\mathcal{L}_{cycle}(G, F) = \mathbb{E}_{x \sim P_X}\left[ \|F(G(x)) - x\|_1 \right] + \mathbb{E}_{y \sim P_Y}\left[ \|G(F(y)) - y\|_1 \right]$$

**身份损失（Identity Loss）**（可选，用于保持色调）：
$$\mathcal{L}_{identity}(G, F) = \mathbb{E}_{y \sim P_Y}\left[ \|G(y) - y\|_1 \right] + \mathbb{E}_{x \sim P_X}\left[ \|F(x) - x\|_1 \right]$$

身份损失鼓励生成器在接收"自己域"的输入时尽量保持输出不变（例如，将梵高风格图像输入 $G$，期望输出依然是梵高风格），这对照片→风格画等任务特别有效。

**总损失**：
$$\mathcal{L}(G, F, D_X, D_Y) = \mathcal{L}_{GAN}(G, D_Y, X, Y) + \mathcal{L}_{GAN}(F, D_X, Y, X) + \lambda \cdot \mathcal{L}_{cycle}(G, F) + \lambda_{id} \cdot \mathcal{L}_{identity}(G, F)$$

其中 $\lambda$ 通常取 10，$\lambda_{id}$ 通常取 0.5（或设为 $\lambda/2$）。

### 3.4 网络架构

**生成器**：CycleGAN 使用类似 Johnson 等人的感知损失的残差块架构：
- 3 个卷积层进行下采样；
- 6 或 9 个 ResNet 残差块（取决于输入分辨率，如 $128 \times 128$ 用 6 个，$256 \times 256$ 用 9 个）；
- 2 个转置卷积层进行上采样；
- 使用实例归一化（InstanceNorm）替代批归一化——因为实例归一化对单张图像的对比度归一化更适合风格迁移任务，不受批次统计影响。

**判别器**：使用 $70 \times 70$ 的 PatchGAN（与 pix2pix 相同）。

### 3.5 CycleGAN 的局限

- 对几何形变（如狗→猫）的处理能力有限——循环一致性倾向于保留底层结构；
- 训练在小数据集上不稳定；
- 无法实现多模态输出（给定 $x$ 只生成一种 $y$）。

## 4. 其他重要变体

### 4.1 StarGAN：多域图像翻译

**StarGAN** 由 Choi 等人于 2018 年提出，用一个统一的生成器实现**多个域之间的相互翻译**。核心设计：

- 生成器接受图像和目标域标签作为输入：$G(x, c_{target}) \rightarrow y$；
- 判别器同时输出**域分类**结果（辅助分类器），判断生成图像属于哪个域；
- 一次性训练即可实现所有域之间的翻译，无需为每对域训练独立模型。

StarGAN v2 进一步引入了**风格编码器**和**映射网络**（受 StyleGAN 启发），实现了给定一张输入图像的**多样化输出**。

### 4.2 MUNIT：多模态非配对翻译

**MUNIT（Multimodal Unsupervised Image-to-Image Translation）** 由 Huang 等人于 2018 年提出，解决了 CycleGAN 单模态输出的问题。

核心思想：将图像的隐表示分解为**内容编码（Content Code）**和**风格编码（Style Code）**：
- 内容编码 $c$：域共享的、空间保留的表示（如物体形状和位置）；
- 风格编码 $s$：域特有的、与空间无关的表示（如纹理和颜色）。

训练时，从域 $X$ 提取内容编码 $c_X$，从域 $Y$ 提取风格编码 $s_Y$，然后用生成器组合：$y = G(c_X, s_Y)$。通过双向重建损失和对抗损失进行训练。不同风格编码 $s_Y$ 的采样产生多样的输出。

MUNIT 与 CycleGAN 不同，MUNIT 不要求 $G(F(y)) = y$ 严格成立（因为风格是多模态的），而是使用**隐空间重建损失**确保内容编码的一致性。

### 4.3 UNIT：共享隐空间翻译

**UNIT（Unsupervised Image-to-Image Translation）** 基于"两个域的图像可以被映射到同一个共享隐空间"的假设。使用变分自编码器（VAE）框架和权重共享，强制编码器在源域和目标域之间学习共享的隐表示。

### 4.4 GauGAN / SPADE

**SPADE（Spatially-Adaptive Normalization）** 由 Park 等人于 2019 年提出（NVIDIA GauGAN 的基础），实现了对语义布局的高质量可控生成：

$$\text{SPADE}(x, seg) = \gamma(seg) \cdot \frac{x - \mu(x)}{\sigma(x)} + \beta(seg)$$

其中 $\gamma(seg)$ 和 $\beta(seg)$ 是从语义分割图 $seg$ 学习到的空间自适应调制参数，在**逐空间位置**对归一化后的特征进行缩放和平移。与 BN 使用全局统计量不同，SPADE 使语义信息在各个分辨率层上都得到有效注入，避免了语义信息的"洗掉"。

### 4.5 Contrastive Unpaired Translation (CUT)

**CUT** 将**对比学习**引入非配对图像翻译，替代 CycleGAN 的循环一致性损失。核心思想是利用**PatchNCE 损失**：输入图像中同一位置的 patch 应该与对应输出位置的 patch 共享相似的嵌入表示（正例），而与其他位置的 patch 不同（负例）。CUT 只需单向生成器，训练速度更快且内存占用更低。

## 5. 总结对比

| 模型 | 年份 | 配对需求 | 域数量 | 多模态 | 核心机制 |
|------|------|---------|--------|--------|---------|
| cGAN | 2014 | — | 通用框架 | 否 | 条件拼接 |
| pix2pix | 2017 | 配对 | 2 | 否 | cGAN + L1 + U-Net + PatchGAN |
| CycleGAN | 2017 | 非配对 | 2 | 否 | 循环一致性 |
| StarGAN | 2018 | 非配对 | 多个 | 否 | 辅助分类器 |
| MUNIT | 2018 | 非配对 | 2 | 是 | 内容-风格解耦 |
| SPADE/GauGAN | 2019 | 配对 | 2 | 否 | 空间自适应归一化 |
| StarGAN v2 | 2020 | 非配对 | 多个 | 是 | 风格编码 + 映射网络 |
| CUT | 2020 | 非配对 | 2 | 否 | 对比学习 |

从 cGAN 的条件注入，到 pix2pix 的配对翻译范式，再到 CycleGAN 的非配对循环一致性，以及后续的多域、多模态和空间自适应方法，图像翻译领域在灵活性、可控性和生成质量上持续突破。
